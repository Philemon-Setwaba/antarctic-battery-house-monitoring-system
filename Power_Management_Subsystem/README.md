# Power Management Subsystem

This folder contains files related to the power management subsystem of the Smart Battery Thermal Regulation System.

## Purpose

This subsystem is responsible for managing and monitoring the power supplied to the system. It supports safe operation by regulating voltage, monitoring battery condition, and supplying power to the required subsystems.

## Contents

This folder may include:

- Circuit design files
- Battery monitoring files
- Voltage regulation files
- Current and voltage sensing files
- Simulation files
- Testing evidence
- Supporting documentation

## Subsystem Role

The power management subsystem ensures that the system receives stable and reliable power while monitoring key battery information such as voltage, current, and state of charge.
