# Power Harvesting Subsystem

This folder contains files related to the power harvesting subsystem of the Smart Battery Thermal Regulation System.

## Purpose

This subsystem is responsible for supplementing the battery supply by harvesting energy from the environment. The design uses a micro wind-based energy harvesting approach to support the power demands of the system.

## Contents

This folder may include:

- Turbine design files
- Piezoelectric disc configuration files
- Rectifier circuit files
- Energy storage files
- Mechanical design files
- Testing evidence
- Supporting documentation

## Subsystem Role

The power harvesting subsystem provides an additional energy source to help extend system operation and reduce complete dependence on the battery.
