# Smart Battery Thermal Regulation System

This repository contains the project files for the EEE4113F Smart Battery Thermal Regulation System. The project focuses on developing a prototype system for improving the reliability of battery-powered Antarctic wave sensors by monitoring battery conditions, regulating heating, harvesting energy, and communicating system data to a user interface.

# Project Purpose
The purpose of this project is to demonstrate a prototype smart battery thermal regulation system that can support remote Antarctic sensing applications. The system aims to reduce the risk of battery failure in cold environments by combining heating, monitoring, energy harvesting, and remote communication.
