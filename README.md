# Antarctic Battery House Monitoring System

This project implements a communication subsystem for remotely monitoring and controlling an Antarctic battery house. The system uses ESP32-based LoRa communication, a Python serial bridge, a FastAPI backend, a PostgreSQL database, and a web-based user interface.

## Features

- Remote telemetry transmission using LoRa
- Bidirectional command control
- Heater and power mode control
- FastAPI backend for data handling
- PostgreSQL database storage
- Web dashboard for live monitoring
- CSV telemetry download functionality
