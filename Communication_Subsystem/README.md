# Communication Subsystem

This folder contains files related to the communication subsystem of the Smart Battery Thermal Regulation System.

## Purpose

This subsystem is responsible for sending telemetry data from the remote unit to the user interface and allowing control commands to be sent back to the remote unit.

## Contents

This folder may include:

- Remote unit code for the Arduino Uno R3
- Base station receiver code for the ESP32 Dev Board
- Python serial bridge script
- FastAPI backend files
- PostgreSQL database scripts
- Web-based dashboard files
- Supporting documentation

## Subsystem Role

The communication subsystem provides telemetry transmission, bidirectional command handling, database storage, dashboard updates, communication warnings, and CSV data export.
